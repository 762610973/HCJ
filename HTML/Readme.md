# HTML 简单笔记
- 标签：单标签、双标签
- 双标签关系：包含关系、并列关系

```html
<html> 根标签
    <head>文档头部
    <title></title>
    </head>
    <body>主题内容
    </body>
</html>

以下是初始的文件模板，暂做保存

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>#[[$Title$]]#</title>
</head>
<body>
#[[$END$]]#
</body>
</html>


```
- 属性标签不分先后，属性和属性之间以空格分开
- 属性采用键值对的格式
